# Q1.[html] [DOM和BOM有什么区别？](https://github.com/haizlin/fe-interview/issues/110)
# A1：
* DOM是文档结构模型， 平时我们使用的对HTML内容进行增删查改的都是DOM的方法
* BOM是浏览器对象模型，它包含了一些浏览器内置的对象，例如常用的location，navigator，history等等
