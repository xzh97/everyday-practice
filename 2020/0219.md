# Q3.[js] [请写出一个函数求出N的阶乘（即N!）](https://github.com/haizlin/fe-interview/issues/191)
# A3：
```javascript
    function getNum(val){
        let result = 1;

        for(let i = 1; i<= val; i++){
            result *= i
        }
        return result;
    }

```
