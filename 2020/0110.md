# Q2.[css] [说下line-height三种赋值方式有何区别？](https://github.com/haizlin/fe-interview/issues/115)
# A2：
* 带单位：先计算成px再计算。例如 1.5em、15px，
* 不带单位：倍数，1.5，子元素行高 = 子元素`font-size` * `line-height`。
